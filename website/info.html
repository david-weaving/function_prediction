About the Function Predictor

This app is a function predictor. It takes in 6 (x,y) points. The points are passed through an AI model that determines the function type: polynomial, exponential, sine, or natural log. Then, the points are fitted to the function type predicted through mathematical algorithms. Once fitted, we can generate the rest of the points as well as returning the function itself.

Please note that this application is in early access and the AI model may get some points wrong. I am always looking to improve the model as well as the website for the best user experience.
